LEADERSHIP ONLINE
- Added distinct role icons for Guild Leader, Raid Leader, Officer and Helper.

FULL ACTIVITY WINDOWS
- Added a full Guild Activity modal opened from Home -> Recent Useful Activity -> View All.

PROFESSION NAVIGATION AND FILTERING
- Added dynamic category tabs that change with the selected profession.
- Added slot-based Enchanting categories: Weapon, Chest, Bracers, Gloves, Boots, Cloak, Shield, Legs and Other.
- Added practical category sets for Alchemy, Blacksmithing, Tailoring, Leatherworking, Engineering, Cooking, Jewelcrafting and Mining.
- Added required-level filters: Any, 1-20, 21-40, 41-59, 60 and Unknown.
- Added rarity filters: Any, Common, Uncommon, Rare, Epic and Unknown.
- Added sorting by online availability, name, required level, rarity, recent scan and crafter count.
- Added Online: All / Only filtering.
- Reduced visible recipe rows to eight to reserve a stable filter area without overlap.
- Preserved recipes with unavailable item-cache data under Unknown.

CRAFTING METADATA
- Extended local profession scans with item level, required level, item type, subtype and equipment slot when cached.
- Corrected item texture extraction to use the Vanilla GetItemInfo texture return.
- Restored cached remote metadata when an item ID is available locally.

WORLD CHANNEL
- Added automatic World-channel detection by channel name and the current joined-channel list.
- Recruitment now displays AUTO /n with the real client-assigned channel number.
- World send buttons use the detected number rather than assuming /5 or /6.
- Kept a clearly labelled manual fallback when automatic detection is unavailable.
- Refreshes detection on login, entering the world, channel notices and Recruitment refresh.

DIAGNOSTICS AND DATA
- Advanced SavedVariables schema from 7 to 8.

- Added diagnostics for World channel, useful activity count and active profession filters.
