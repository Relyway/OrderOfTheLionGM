# Security and data handling

OrderOfTheLionGM exchanges addon messages only inside the game client and stores guild-related state in local SavedVariables.

Please do not publish:

- account names or private contact information;
- personal SavedVariables or complete `WTF` folders;
- officer notes or private guild exports;
- credentials, access tokens, or private repository keys.

For a security-sensitive report, contact the maintainer privately through the contact information listed in the addon metadata instead of opening a public issue.
