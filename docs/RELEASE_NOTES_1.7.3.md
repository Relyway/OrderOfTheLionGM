# OrderOfTheLionGM 1.7.3

Launcher compatibility release for OctoWoW.

## Main changes

- The live addon files now reside directly in the repository root.
- OctoLauncher can install the addon through the custom Git addon URL.
- Future stable updates can be installed through Update or Update All.
- Automatic GitHub validation and install ZIP generation remain available.
- Addon functionality and SavedVariables remain compatible with 1.7.2.

## Compatibility

- Interface: 11200
- SavedVariables schema: 14
- Network protocol: 3
- No WTF or SavedVariables reset is required.

## OctoLauncher installation

Remove the previous non-versioned copy of OrderOfTheLionGM, then add this custom Git addon:

https://github.com/Relyway/OrderOfTheLionGM

Do not delete the WTF folder.
